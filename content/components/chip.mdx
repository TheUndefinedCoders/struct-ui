## **Props**
Props that can passed to Chips Component.
| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **`chipData`** | `IChip` | Required | An object containing the chip's data like `label` and `id`. |
| **`border`** | `'left' \| 'right' \| 'top' \| 'bottom'` | `undefined` | Specifies which side of the chip should have a border. |
| **`borderColor`** | `string` | `undefined` | Sets the color of the border. |
| **`textColor`** | `string` | `'#2D3748'` | Sets the text color of the chip. |
| **`bgColor`** | `string` | `'#FFFFFF'` | Sets the background color of the chip. |
| **`isRounded`** | `boolean` | `true` | If `true`, the chip will have fully rounded corners. |
